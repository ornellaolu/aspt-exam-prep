🔐 How Access Codes Work

1. System Overview

When a student purchases a course, you:

  1. Generate a unique access code (e.g., PHLEB-12345)
  2. Add it to codes.json with what they purchased
  3. Send the code to them via email
  4. They enter it on access.html to unlock their courses

2. Creating Access Codes

Format: TRACKTYPE-RANDOMNUMBER

Examples:

PHLEB-50298 (for Phlebotomy only) EKG-49201 (for EKG only) PCT-51832 (for PCT only) CMA-48921 (for CMA only) BUNDLE-50102 (for all 4 tracks)

3. Adding Codes to codes.json

When a student purchases Phlebotomy ($99):

"PHLEB-50298": { "access": ["phlebotomy"], "studentName": "John Smith", "purchased": "2026-06-18" }

When a student purchases Bundle ($249):

"BUNDLE-50102": { "access": ["phlebotomy", "ekg", "pct", "cma"], "studentName": "Jane Doe", "purchased": "2026-06-18" }

🚀 Deployment to Vercel

Step 1: Prepare Your Files

Make sure all files are in /Users/ornellaoluwole/ASPT_Website/

  • index.html
  • access.html
  • dashboard.html
  • codes.json
  • courses/ folder (with your course files)
Step 2: Buy a Domain

Go to Namecheap, GoDaddy, or Route53 and buy a domain. Suggestions:

  • asptcertified.com
  • asptmasterclass.com
  • asptexamprep.com
  • phlebotomyexamprep.com

Cost: $10-15/year for most domains

Step 3: Create Vercel Account
  1. Go to vercel.com
  2. Sign up with GitHub, GitLab, or email
  3. Click "Import Project"
  4. Point to your ASPT_Website folder (via GitHub or direct upload)
  5. Deploy!

Cost: Free (generous free tier)

Step 4: Connect Domain
  1. In Vercel: Settings → Domains
  2. Add your domain
  3. Update DNS records (Vercel will show you how)
  4. Wait ~24 hours for DNS to propagate

🔄 Updating Codes After Deployment

After you deploy to Vercel:

  1. Make changes to codes.json locally
  2. Commit and push to GitHub (or re-upload to Vercel)
  3. Vercel auto-deploys in ~2 minutes
  4. Students can use new codes immediately
⚠️ Important: Each time you add a code, you must commit/push the change. Vercel won't automatically sync local file changes.

💡 Important Notes

✓ Static Site Benefits:
